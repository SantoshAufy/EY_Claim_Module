# Etihad Insurance Claim Portal - Fixed Clickable Prototype

Open `index.html` in any modern browser.

This fixed build embeds the CSS and JavaScript directly in the HTML so the prototype does not open as an unstyled page if files are previewed incorrectly. The icons and brand assets are still included under `assets/` and referenced by the UI.

Clickable flow included:
- Home dashboard
- Claim type selection popup
- My Claims
- Documents & Forms
- Raise New Claim for Life Insurance, Pilot Income Protection, Personal Accident, and Critical Illness
- Fetch employee details, upload simulation, save draft, submit claim, support, filters, and role toggle
